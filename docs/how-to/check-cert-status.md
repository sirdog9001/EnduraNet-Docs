To see the certification status of a member click the `Search...` dropdown and select either `By Member` or `By Certification`. 

On either page the thing to look out for will be 2 columns:

- Certified on Version
- Expires On

*Both* columns must have green (ğŸŸ©) text in order for the certification to be valid. If *any* column is red (ğŸŸ¥) that means the certification is *no longer valid* and the member should be treated as not having it.
